关于时间胶囊 (HelloTime)

Spring 全栈演示

这一套实现使用 Spring Boot MVC + Thymeleaf + HTMX， 展示服务端模板渲染、会话态管理与局部 HTML 片段更新如何在同一应用中协同工作。

与其他 JavaScript 全栈实现不同，这里由服务器主导页面状态，HTMX 仅负责在局部交互时提升流畅度。

☕

服务端优先

控制器直接返回模板页面，首屏即带完整内容与状态，无需等待客户端首次拉数。

🧩

HTMX 局部更新

管理员列表、分页与删除操作使用 HTML 片段替换，不必引入完整前端框架。

🗃️

持久化内聚

应用、页面、API 与 SQLite 数据层都封装在同一 Spring Boot 进程中。
