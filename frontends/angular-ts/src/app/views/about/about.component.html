项目简介

时间胶囊 (HelloTime) 是一个类似 RealWorld 的技术展示应用。 通过统一的 API 规范和可复用的前端样式，展示不同前后端技术栈的组合能力。

当前技术栈

  • 前端框架: Angular 18 + TypeScript
  • 后端框架: {{ techStack()?.framework || '加载中...' }}
  • 后端语言: {{ techStack()?.language || '加载中...' }}
  • 数据库: {{ techStack()?.database || '加载中...' }}
  • 样式系统: 共享 CSS Design Tokens
  • 状态管理: Angular Signals

设计理念

  • 统一的 REST API 规范 (OpenAPI 3.0)
  • 可复用的前端样式系统 (CSS 自定义属性)
  • 支持亮色 / 暗色主题切换
  • 前后端分离，可自由组合技术栈
  • Angular Standalone Components，无 NgModule
